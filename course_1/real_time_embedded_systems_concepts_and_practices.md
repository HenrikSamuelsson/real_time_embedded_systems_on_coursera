# Real-Time Embedded Systems Concepts and Practices

Real-Time Embedded Systems Concepts and Practices is the first course in the Real-Time Embedded Systems Specialization.

Learnings:

- Basic understanding of rate monotonic theory, analysis, and practices for embedded systems.
- Implementation of real-time services as POSIX threads.
- Difference between Linux user space and kernel space.
